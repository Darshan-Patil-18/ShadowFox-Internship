# ShadowFox Data Science Internship

This repository contains the projects and tasks completed as part of the **ShadowFox Data Science Internship**.
The internship focuses on developing practical skills in **data analysis, visualization, and exploratory data analysis (EDA)** using Python.

During the internship, I worked on multiple datasets and applied visualization techniques to extract insights and understand data patterns.

---

# Internship Tasks

## Task 1 – Visualization Library Documentation

In this task, I explored two important Python visualization libraries:

* **Matplotlib**
* **Seaborn**

The goal was to understand how different types of graphs are created and when they are used in data analysis.

### Graphs Implemented

* Line Plot
* Scatter Plot
* Bar Chart
* Histogram
* Pie Chart
* Box Plot
* Violin Plot
* Heatmap
* Count Plot
* Pair Plot

This task helped build a strong foundation in **data visualization techniques used in exploratory data analysis**.

---

## Task 2 – Delhi Air Quality Analysis

This project performs **exploratory data analysis on air pollution data from Delhi**.
The dataset contains measurements of pollutants such as PM2.5, PM10, CO, NO₂, SO₂, and others.

### Key Analysis

* Distribution of pollutant concentrations
* Correlation between pollutants
* Hourly pollution trends
* Daily pollution patterns
* Comparison of average pollutant levels

### Key Insight

Particulate pollutants such as **PM2.5 and PM10 show strong correlation**, indicating they may originate from similar pollution sources.

---

## Task 3 – Amazon Product Ratings and Reviews Analysis

This project analyzes **Amazon product ratings, prices, discounts, and reviews** to identify patterns in customer feedback and product popularity.

### Key Analysis

* Distribution of product ratings
* Relationship between discounts and ratings
* Identification of most reviewed products
* Category distribution analysis

### Key Insight

Most products have ratings between **3.5 and 4.5**, showing generally positive customer feedback.
