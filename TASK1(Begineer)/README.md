# Task 1: Visualization Library Documentation  
**ShadowFox Data Science Internship**

In this task, I explored and documented two popular Python visualization libraries: **Matplotlib** and **Seaborn**.

The goal was to understand the variety of graphs each library can generate and how they are used in real-world data analysis.

---

## Libraries Covered
- **Matplotlib** → Core Python plotting library with full customization  
- **Seaborn** → Built on Matplotlib, best for statistical and attractive visualizations  

---

## Graphs Implemented
The notebook includes examples of:

- Line Plot  
- Scatter Plot  
- Bar Chart (Vertical & Horizontal)  
- Histogram / Histplot  
- Pie Chart  
- Box Plot  
- Violin Plot  
- Heatmap  
- Count Plot  
- Pair Plot  

Each plot contains a short explanation, use case, and working code.

---

## Quick Comparison

| Feature | Matplotlib | Seaborn |
|--------|------------|---------|
| Ease of Use | More manual | Simple & beginner-friendly |
| Styling | Basic by default | Beautiful by default |
| Statistical Plots | Limited | Strong (heatmap, violin, pairplot) |
| Best For | Full customization | Quick EDA and insights |
